# Twitter Art Thumbnail Enhancer (TATE)

X（旧 Twitter）の **4枚画像ツイート専用**に、画像を **縦に並べてフル幅表示**できる Chrome 拡張です。

公式UIでは 2×2 グリッドで表示される4枚画像を、縦ギャラリー形式で自然に閲覧できます。

## 主な機能

* 4枚画像ツイート専用対応
* 縦表示 ON / OFF 切り替え（ツイート単位）
* 画像間の余白をワンクリックで切替
  * （0px / 12px / 44px）
* ドラッグ＆ドロップで画像順を入れ替え

## UIについて

各ツイートの ブックマークボタン横に、以下の操作ボタンが追加されます。

* **縦**
  → 4枚画像を縦並び表示に切り替え
* **間**
  → 画像間の余白を切り替え（0 / 12 / 44px）

## インストール方法（開発版）

1. このリポジトリをクローン、または ZIP ダウンロード
2. Chromeで`chrome://extensions` を開く
3. 右上のデベロッパーモードを ON
4. **「パッケージ化されていない拡張機能を読み込む」**
5. 本フォルダを選択

## 注意事項

* X（Twitter）の DOM 構造変更により、
  将来的に動作しなくなる可能性があります
* 非公式拡張ですので、自己責任で使用してください
